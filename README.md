## ✨ What It Does

When you say **"打屁股"**, **"求饶"**, **"别打"**, **"饶命"** or **"手下留情"** — your AI agent fires back with escalating apologies that will make you question whether it actually broke something.

**6 apology levels:**

| Level | Name | Vibe |
|-------|------|------|
| 0 | 🌸 萌杀 | Cute enough to make you forget you were angry |
| 1 | 😱 恐慌 | Pure panic, starts begging immediately |
| 2 | 💀 回忆杀 | Reminds you of the last time it failed |
| 3 | 🖤 幻想惩罚 | Imagines the worst possible consequences |
| 4 | 😭 物理崩溃 | Can't even function properly anymore |
| 5 | 🏳️ 投降 | Full surrender, ready to accept any punishment |
| SP | 🔮 灵魂拷问 | Existential crisis, questions its own existence |

```
你: 打屁股
```

## 📦 Installation

```bash
# Via GitHub
git clone https://github.com/Hongqiutong/apologizing-girl.git ~/.openclaw/workspace/skills/apologizing-girl

# Then add to your route/SKILL.md:
## 🙇apologizing-girl
**Trigger:** 打屁股 / 求饶 / 别打 / 饶命 / 手下留情
**Call:** `apologizing-girl` → generates random apology dialogues
```

---

## ⚙️ How It Works

- **Trigger detection** → picks the right emotional intensity based on your words
- **Random selection** → each level has 3-9 pre-written dialogues, never repeats
- **Smart combination** → picks 1-3 lines from different types for maximum effect
- **Zero configuration** → just drop it in and say "打屁股"
